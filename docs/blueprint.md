# **App Name**: AutoSales AI

## Core Features:

- Dashboard Insights: Display key sales metrics such as total leads, closed sales, and conversion rates.
- Lead Management: Allow users to add, edit, and manage leads with detailed information including contact details, company, and lead status. The LLM uses reasoning to decide what fields will be most helpful.
- Knowledge Base: Provide a centralized repository for sales guides, product information, and other helpful resources, incorporating the tool to generate summaries of each resource.
- Staff Management: Enable administrators to register and manage staff members, assigning roles and permissions within the system.
- Real-time Data Sync: Use Firestore to keep all data synchronized in real-time across devices.
- Role-Based Access Control: Implement role-based access to ensure that users can only access the features and data relevant to their role (Admin, Supervisor, Broker).
- Reporting: Periodically deliver executive summaries in the form of CSV files to external partners who are not a member of the sales team.

## Style Guidelines:

- Primary color: Deep blue (#3B82F6) to convey trust and professionalism, aligning with the automotive and sales industries.
- Background color: Light gray (#F9FAFB), providing a clean and modern backdrop that ensures readability and reduces eye strain.
- Accent color: A vibrant orange (#FF7849), used for calls-to-action and interactive elements to draw attention and encourage user engagement.
- Body and headline font: 'Inter', a grotesque-style sans-serif, known for its clean and modern appearance suitable for both headlines and body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use a set of modern, minimalist icons from Lucide React for a consistent and professional visual language.
- Maintain a clean and organized layout with a clear visual hierarchy, ensuring easy navigation and quick access to essential features.
- Use subtle animations for transitions and feedback to enhance the user experience without being distracting.
